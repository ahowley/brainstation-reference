Node

Node is an interpreter built in C++ that allows us to run JavaScript in the terminal and create full-stack applications with one programming language. Learn more about node on node's website here.

Overview

  • Node is built on a package system that uses two commands: npm and npx.
  • To install a node package, we use npm install. For example, to install Sass, we can use npm install sass.
  • Another example would be axios - we can install axios with npm install axios.
  • Each time we install a package with npm, it automatically updates our package.json and package-lock.json with information about what we installed and what version we are on.
  • To install what's in package.json, we can run npm install or npm i.
  • Like many other interpreted languages, Node provides a REPL (Read-Eval-Print-Loop) for JavaScript. To run this in our terminal with node installed, we can simply type node.
