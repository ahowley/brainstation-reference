Array Methods

Arrays in JavaScript have several very useful built-in functions we can leverage to manipulate them in more efficient ways than just using a traditional for loop each time to accomplish our goal. Many of them are higher order functions, which means they often accept a function as an argument, which will run once per item in the array.

.forEach() and .map()

  • Arrays' built-in [].forEach(elem => { //do stuff with each elem }) method provides an easy way to apply a function to each element of an array.
  • To use this function, the function we provide as an argument to .forEach() needs a parameter that can accept each item in the array one at a time.
  • Below is an example that uses forEach to add up all the numbers in an array and logs the result. 
    // create a variable equal to an array of numbers
const arrayOfNumbers = [2, 3, 3, 2]; // The sum of 2 + 3 + 3 + 2 is 10

// Create a variable to store our sum in
let sumOfNumbers = 0;

// Pass each number in arrayOfNumbers to a function as an argument, one at a time
arrayOfNumbers.forEach((number) => {
  // Add the current number to the total sumOfNumbers
  sumOfNumbers += number;
});

// Print the result to the console
console.log(sumOfNumbers);
  • Arrays' built-in [].map(elem => { //do stuff with each elem } is exactly like forEach, except it will also save the result of each iteration of the function inside a new array.
  • .forEach() does not return anything, while .map() returns a new array. Otherwise, these are largely the same, in that they invoke a function for each element in the array.

Other Array Methods

  • [].find((elem) => {}) will find and return the first elem that causes the function provided to return true.
  • [].filter((elem) => {}) works just like .find(), except it returns an array, so it can return multiple matches. This is like the difference between .querySelector() vs. .querySelectorAll().
  • [].reduce((current, accumulator) => {}, startValue) will reduce an array down to just one value. This one's tough to wrap your head around if it's new to you, but to help, here's the same example we used for .forEach(), but using .reduce() instead: 
    // create a variable equal to an array of numbers
const arrayOfNumbers = [2, 3, 3, 2];

// Pass each number in arrayOfNumbers to a function as an argument, one at a time
arrayOfNumbers.reduce((sumOfNumbers, number) => {
  // Add current number to the total sumOfNumbers
  sumOfNumbers += number;
  
  // Return the current sumOfNumbers each time the function runs so it adds up across iterations
  return sumOfNumbers;
}, 0 // set sumOfNumbers's starting value - equivalent to "const sumOfNumbers = 0;" from .forEach() example
);
  • There are lots of other handy methods - check out MDN page on the Array object to see all of them!
