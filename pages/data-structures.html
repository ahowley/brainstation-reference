Data Structures

Data Structures are various types of ways to organize data into neat little packages - your "moving boxes" of software engineering. The two types of data structures we'll be using most prominently are JavaScript's Arrays and Objects.

Arrays

  • For an exhaustive list of Array methods and parameters, see the MDN page here.
  • An array is a mutable, ordered collection of values - each value has an index in the array that we can use to find it.
  • The first item in a JavaScript array has an index of 0, which means JS Arrays are zero-indexed.
  • We can declare a variable and assign an array to it in a couple of ways. The first is using the Array object constructor: const arrayVar = new Array(); or using the Array literal: const arrayVar = [];. It is much, much more common to use the literal.
  • Arrays in JavaScript are not sized - items can be added to or removed from an Array at will.
  • An Array in Javascript isn't typed - it can hold values of any type, including other arrays.
  • We can provide values inline when defining an array: let arrayVar = ["Item 1", 2, 3.5, 176]
  • To access a value at a specific index, we again use brackets: arrayVar[0]; // "Item 1"
  • To add an element or element(s) to the end of an array (and return the length of the array afterward), we use the push method: arrayVar.push("Item 4"); or arrayVar.push("Item 4", "Item 5");
  • To add an element or element(s) to the start of an array (and return the length of the array afterward), we use the unshift method: arrayVar.unshift("New Item 1"); or arrayVar.unshift("New Item 1", "New Item 2");
  • We can extend an array with the values from another array using the spread operator: arrayVar.push(...["Item 5", "Item 6"]);
  • To remove an element out of the end of an array (and return the element removed), we can use the pop method: arrayVar.pop(); // "Item 6"
  • To remove an element out of the start of an array (and return the element removed), we can use the shift method: arrayVar.shift(); // "Item 1"
  • To target a specific index and remove a certain number of elements from that location, we can use the splice method: arrayVar.splice(1, 2);
  • To know how many elements are in an Array, we can use the length property (note this is not a method): arrayVar.length;

Objects

  • For an exhaustive list of Object methods and parameters, and an overview of the data structure in JS, see the MDN page here.
  • An Object in JavaScript is a collection key-value pairs.
  • Almost everything in JavaScript is an object - JS is fundamentally object-oriented.
  • We can declare a variable and assign an Object to it similarly to arrays: const objectVar = new Object(); or let objectVar = {}
  • We can assign values when an object is created similarly to Arrays as well: const objectVar = { key1: "value1", key2: "value2" }
  • When we leave out quotation marks when defining a key, that key will be defined as a string.
  • We can add values to an object with brackets, similarly to Arrays, or with dot notation: objectVar["value3"] = 3; or objectVar.value3 = 3;
  • We access values and/or properties in an object just like we do with Arrays, with brackets containing the key/"index", or we can access them with "dot notation": objectVar.value3; // 3
  • There can only be one value assigned to a key at a time - if we assigned a new value to a key, it will overwrite the old value.
  • An Object's key is always a String.
