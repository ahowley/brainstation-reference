Flexbox Quick Reference

  • For flex magic to work, we first need a parent container with display: flex
  • display: flex makes a container, elements inside the container can be "directed" (literally) using flex-direction
  • justify-content and align-items set on the flex container will manage aligning content along the main and cross axis, respectively.
  • flex-wrap is used to define what happens when the width or min-height of the parent container or screen is too narrow/short to display all children in a flex container.
  • align-self can be used in a child element to independently change only 1 child's cross-axis alignment.
  • order in a child element allows us to specify the display order of children elements.
  • flex-grow lets us define how much space a child element will occupy along the main axis, proportional to other children.
