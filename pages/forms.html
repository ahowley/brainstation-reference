HTML Forms

HTML Forms allow us to collect input from a user an pass that input to be processed, on either the frontend or the backend (or both). See the MDN page for more details!

  • We can create a <form> element with multiple <input> elements, which can have a variety of types.
  • Form elements accept an action and a method. The method is an http method, like GET, POST, etc. The action tells the form what URL should be used to process the form. When method="dialog" is set, the action is ignored.
  • <input> elements can have a type, which allow us to dictate how a form should validate that field. For example, type="email" ensures the value submitted from that field is a valid email-like string (by default).
  • <input> fields also require a name attribute, which is the "key" part of that fields "key-value pair" when it is passed to JavaScript. This is especially useful when making use of the FormData() constructor or the 'formdata' event (see here for more).
  • Along with input elements, there are various other input-like fields made for use with forms, including (but not limited to) <select> and <option>s, <textarea>, and <button>
  • If we give a form input-like field an id, we can also give it an associated <label> element anywhere else in our html (usually right before or right after, as a sibling). We can then give the label an attribute that specifies the element ID that the label is for, allowing events on the label like users clicking on it to select the form field. For example: 
    <label for="email">Enter your email</label>
<input type="email" name="email" id="email">
  • Forms have a few special events associated with them: probably the most important is 'submit'. We could add an event listener that runs on form submit like so: document.getElementById("form").addEventListener('submit', callback);
  • Forms also trigger a 'formdata' event whenever a FormData object is made out of them. MDN mentions that this happens on submit as well (by default), but we can trigger this event manually like so: 
    new FormData(document.getElementById("form"));
  • To set or get the value of an input field, we can use the .value property of that element. 
    document.querySelector('input').value = 'new value';
    We can also set the value in HTML: 
    <input name="key" value="value">
  • We can specify that a field is required using the required attribute in the field's html tag. Note that we don't set this attribute to any value - simply including it labels the field as required.
