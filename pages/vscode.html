VSCode Quick Reference

Hotkeys listed are the Windows (and Linux, I think?) defaults - if you're on Mac, replacing "ctrl/control" with "cmd/command" and replacing "alt" with "option" should make these work!

VSCode

  • General
    • ctrl + shift + palette to open the Command Palette - sort of a "search anything everywhere" button
    • ctrl + keyboard ctrl + shortcuts to view all hotkeys
  • Editing/selection
    • tab to auto-complete what you're currently typing
    • shift + tab to reduce current line/selection indentation by 1 level (basically reverse tab)
    • ctrl + line to select current line
    • alt + click (and drag) to add another cursor/selection to your current cursor/selection, allowing you to edit multiple lines at once
    • ctrl + duplicates to find and add the next instance of whatever you have selected to your selection - ctrl+f's next-door neighbor, great for quickly renaming a class or variable
  • Comments
    • ctrl + / to comment out (and back in) current line(s)
    • ctrl + shift + a to comment out (and back in) current selection
  • Navigation
    • ctrl + left arrow and ctrl + right arrow to jump around brackets and parentheses and spaces
    • ctrl + shift + left arrow and ctrl + shift + right arrow to select around brackets and parentheses and spaces
    • alt + left arrow and alt + right arrow to jump to your previous/next cursor position - like ctrl+z for your cursor
  • Other
    • ctrl + alt + right arrow to split your current file to the right half of the editor (ctrl + alt + left arrow to re-stack your windows)
    • ctrl + jerminal to open and close an in-editor jerminal

Emmet

This comes default with VSCode, so you shouldn't need an extra plugin. I tried to enter the exact text you'll type, which is why some closing quotations/brackets are missing.

  • ! enter to auto-generate html boilerplate
  • (element) tab to auto-generate an html element. For example, p tab will generate an empty paragraph and put your cursor between tags to add content.
  • (html element)[attribute tab to generate an html element with an empty attribute. For example, div[class tab will generate a div element with class="" and a closing tag, with your cursor btween the quotes to add classes
  • (html element)[attr=value atr="value with spaces" tab to generate an html element with various attributes set to values. For example, div[id=dog class="golden retriever tab will generate a div element with an id value of "dog" and classes "golden" and "retriever", leaving your cursor between opening and closing tags to oadd content
  • (html element).classname tab to create an element with class "classname". For example, div.golden tab generates a div element with class "golden". Can be chained, i.e. div.golden.retriver tab
