Quick Tips

  • Chrome
    • f12 opens dev tools
    • ctrl + r refreshes the page
    • ctrl + shift + r "force" refreshes the page, clearing the cache for the site you're on (give it a shot when you face a bug before crawling through your code)
    • ctrl + shift + c while in the debugger turns on the hover inspection tool
